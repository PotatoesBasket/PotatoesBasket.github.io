Do the Thing is a WarioWare-inspired game that was created for AIE's Game Development course.

Current Features

  • 5 minigames
  • Gameplay that speeds up as you progress
  • Save your score (download ver only)

Planned Features

  • Over 50 minigames
  • Various modes that change up the game speed, lives, and even goals
  • Actual fun (maybe)
